# Tales of Arcana - Races Compendium

A comprehensive Foundry VTT module containing **222 D&D 5e races** from the Tales of Arcana Race Guide.

## Features

- **222 Complete Races** - All races from the Tales of Arcana source material
- **Ability Score Modifiers** - Intelligently assigned based on race type
- **Movement Speeds** - Walk, fly, swim, climb as applicable
- **Languages** - Race-specific languages for each race
- **Traits** - Racial heritage descriptions
- **Easy Import** - Drag and drop from compendium to character sheets

## Installation

### Method 1: Via Foundry Module Manager
1. In Foundry, go to **Add-on Modules**
2. Click **Install Module**
3. Paste this manifest URL: `[Your GitHub Release URL]/module.json`
4. Click **Install**

### Method 2: Manual Installation
1. Download the latest release
2. Extract to your `Data/modules/` folder
3. Restart Foundry
4. Enable the module in your world settings

## Usage

1. After installing the module, go to **Compendium Packs**
2. Find **Tales of Arcana Races**
3. Click to expand and view all 222 races
4. Click a race to view details
5. Drag the race onto a character sheet to apply it

## Compatibility

- **Foundry VTT**: v12.x - v13.x
- **D&D 5e System**: Compatible with official D&D 5e system

## Race List

Includes races such as:

- Elf (Wood, Dark, Sun, Angel variants)
